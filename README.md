# igor-htxrd-heatmap
# HT-XRD Viewer (BL02B2 sweep データ用)

高温放射光 XRD の sweep データを **縦軸=温度 (K) / 横軸=2θ / 強度カラー** の heatmap で表示し、
CIF から計算した理論 XRD パターン(最大 3 相)と同一 2θ 軸で縦に並べて比較・ピーク帰属するための Igor Pro ツール。

## ファイル

| ファイル | 用途 |
|---|---|
| `HT_XRD_Viewer.pxt` | **これをダブルクリックするだけ**。新規実験が開き、自動でパネルが出ます |
| `HT_XRD_Viewer.ipf` | 同内容のプロシージャ単体 (コード編集・既存実験への取り込み用) |

動作要件: Igor Pro 8 以降 (9 推奨)。フォント Arial。

## 使い方 (パネル上から順に)

### 1. Sweep data
- **Load sweep folder...** で `.dat` フォルダを選択。
- ファイル名の `_sv312.0K_` 部分から温度を自動取得し、**温度昇順**に並べます
  (温度タグの無いファイルは無視。同一温度は +0.01 K ずらして単調化)。
- `.dat` は「2θ 強度」の 2 列テキスト。全ファイル同一 2θ グリッド前提 (BL02B2 sweep 標準)。

### 2. Heatmap
- 図サイズは **cm 指定で固定** (初期値: 幅 8.5 cm = 論文シングルカラム、高さ 6 cm)。
- **Font(pt)** 1 箇所でグラフ全体のフォントサイズを一括指定 (軸ラベル内の個別サイズ指定なし、全て Arial)。
- 2θ / T の表示範囲、カラーテーブル (初期値 **YellowHot**)、log10 強度、カラーバー、z スケールを設定して **Make heatmap**。
- 体裁: `expand=2`, `axThick=0.5`, 軸ラベルは `2θ / º`・`Temperature / K`、カラーバーは枠なし `Intensity / counts`。

### 3. Calculated XRD from CIF
- **λ (Å)** (初期値 0.5。2026A の CeO2 校正値は 0.500351)、**FWHM (°)**、**η** (pseudo-Voigt 混合比、0=Gauss/1=Lorentz)、
  **2θ min/max**、**step (°)** (2θ 分解能) はすべてユーザー入力。
- **Load CIF 1–3...** で最大 3 相。読み込み時に自動計算されます。
- **Calculate + plot patterns** で現在のパラメータで再計算してプロット
  (図の横幅は heatmap と同じ cm 固定、**縦幅は自由**に設定可)。
- **Show peak table** で選択相の「2θ / 相対強度 / 多重度 / hkl」一覧表 → ピーク帰属に使用。
- CIF は VESTA / pymatgen / ICSD / COD 出力を想定 (対称操作 `_symmetry_equiv_pos_as_xyz` または
  `_space_group_symop_operation_xyz` が必要。無い場合は P1 扱い)。占有率対応。

### 4. Combined figure
- Heatmap (上段) + 理論パターン (下段) を **同一 2θ 軸**で縦に並べた 1 枚の図を作成。
- 下段の高さ比率 (CIF area fraction) と全体高さ (cm) を指定。横幅は固定幅と共通。

## 計算の中身と検証

- 強度式は pymatgen `XRDCalculator` と同一:
  - 原子散乱因子 f = Z − 41.78214·s²·Σ aᵢ exp(−bᵢ s²)、s = sinθ/λ (99 元素分の係数を内蔵)
  - I ∝ |F(hkl)|² × LP、LP = (1+cos²2θ)/(sin²θ cosθ)
  - 対称操作で全原子展開 → 全 hkl 球で足すので多重度は自動的に考慮
- ピーク形状: 面積規格化 pseudo-Voigt (FWHM 一定)、最大値 100 に規格化
- 同一アルゴリズムの Python 移植で pymatgen と**小数 4 桁一致**を確認済み:

## 制限事項

- 温度は**ファイル名の sv###K のみ**から取得 (JSON の Temperature 欄は空のため)。昇温・降温が混在しても温度順に並びます。
- Debye-Waller 因子 (Biso) は無視 (CIF に有っても未使用。pymatgen デフォルトと同じ)。
- LP は無偏光式。放射光の偏光補正は未実装 (相対比較・帰属用途では実用上問題なし)。
- I0 (入射強度) 規格化は未実装。必要なら matRaw に対して手動で。
- β-Li₃PS₄ クラスの低対称大セルは反射数が多く、計算に数秒〜十数秒かかります。

## 構成メモ (開発者向け)

- 全データは `root:HTXRD:` 以下。CIF は `root:HTXRD:cif0..2`。
- pxt はプロシージャレコード (PTN003, type=5, CRLF+UTF-8 BOM) のみを持つ最小構成で、
  `AfterCompiledHook` がコンパイル直後にパネルを自動起動。
- コード修正は `.ipf` を編集 → pxt 再生成 (または ipf を直接使用)。
